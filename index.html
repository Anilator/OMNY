If U cannot use flexboxes because of their poor browsers support, just use inline-blox. This page demonstrates their capabilities. U may open this page in any browser and make sure.

The '_ibox' class makes null gaps between inline-blocks in all browsers.

The '_ib' class restores font and line sizes. It also allows 'display: inline-block' on IE7-.

Buttons

Use only 'input'-buttons, not 'button' tag if U wanna get correct button's behavior in IE7. Look at that, IE7 doesn't put gaps between inline-blocks.

Don't set vertican paddings and line-height to buttons. See This article for details.

Note, that Safary adds some own styles to buttons. fortunately, we can use 'appearance' property.

Summary, if U wanna get same styled buttons in all browsers, use 'div' and JS. Sadly, semantic and cross-browsering are not friends. U also may use a 'button' tag, but u must add some JS for its support in IE7.
